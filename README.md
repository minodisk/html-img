# html-img package

Fill image size automatically :satisfied:

### Supported Language

- HTML
- Jade

### Supported url type

- Relative path (Like `images/smaple.png`)
- Absolute path (Like `/images/sample.png`)
- Address in web (Like `http://example.com/images/sample.png`)

### Supported image type

- PNG
- JPG
- GIF
